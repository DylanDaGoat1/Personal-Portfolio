# Dylan Dao — Personal Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript to showcase my skills, projects, and contact information.

---

## Features

- **Responsive design** — mobile-friendly layout with a hamburger navigation menu
- **Smooth scroll navigation** — active nav link highlights as you scroll through sections
- **Scroll reveal animations** — sections animate into view as you scroll down the page
- **About section with tabs** — toggle between Languages, Frameworks, and Education
- **Projects showcase** — hover-reveal cards displaying project details
- **Contact form** — form submissions sent directly to Google Sheets via Google Apps Script

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Running Locally

No build tools or dependencies required.

1. Clone the repository:
   ```bash
   git clone https://github.com/DylanDaGoat1/Personal-Portfolio.git
   ```
2. Open the project folder in VS Code
3. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
4. Right-click `index.html` → **Open with Live Server**
5. The site opens at `http://127.0.0.1:5500`

> The contact form requires an HTTP server to work correctly (not `file://`), so Live Server is recommended over simply opening the file directly.
